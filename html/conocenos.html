Historia

La pizzería EII nace como una propuesta para la evaluacion extraordinaria de la signatura Softwares y Estandares para la Web. En esta página web se intenta mostrar los distintos conocimientos adquiridos durante las diferentes practicas de la asignatura. Para ello se aplicaran los conocimientos de html para realizar las diferentes vistas de la aplicacion, de la misma manera se utilizaran los conocimientos adquiridos de css para mostrar de forma correct y tratar los estilos de las vistas.Por otro lado tambien se utilizara xml para guardar las pizzas que conformaran la carta de la aplicacion, juntos con estos xml se crearan las dtds y xmlSchemas necesarios para si validacion, de la misma forma para mostrar las pizzas se utilizaran transformaciones xslt. En lo referente a la computacion en el cliente se utilizara jquery para la creacion de un reloj para la pagina principal de la aplicacion, el formato json sera utilizado para el intercambio de informacion entre el cliente y el servidor y se utilizara Ajax para leer y mostrar la informacion en dicho formato. Tambien sera usado en lo que concierne a la computacion en el clietne la api de geolocalizacion de html5, y se consumiran los servicios web de Openlayers para mostar la ubicacion de los locales a los clientes. Por otro lado la computacion en elservidor se hara sobre php, se utilizara para validar la informacion del clietne y guardarla en una base de datos mysql y se enviara desde php un mensaje al usuarios de que se ha procesado su pedido correctamente. El funcionamiento de todo lo anterior se probara sobre la máquina local XAMPP, y se utilizaran los validadores de html, css, xml, etc. para comprobar que estan bien formados.
